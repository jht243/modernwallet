## Interactive Tools

The project has full React island calculators — one per hub:
- `src/components/AutoLoanCalculator.tsx`, `MortgageExtrasCalculator.tsx`,
  `RetirementCalculator.tsx`, `InvestmentCalculator.tsx`, `NetWorthCalculator.tsx`,
  `PortfolioCalculator.tsx`, `RentalCalculator.tsx`, `BudgetCalculator.tsx`
- Mounted via `src/components/CalculatorIsland.tsx`
- Route: `/{id}/` (hub) + `/{calculator}/{slug}/` (spokes)
- JSON-LD: `src/lib/jsonld.ts` provides SoftwareApplication + FAQPage builders
